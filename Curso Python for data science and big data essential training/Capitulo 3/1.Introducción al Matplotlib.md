# Introduccion al Matplotlib

````
import pandas as pd
import numpy as np 
import matplotlib.pyplot as plt
df = pd.read_csv(r"C:\Users\steve\OneDrive\Documentos\python\base_datos_2008.csv", nrows=100000)
data = np.unique(df.Cancelled, return_counts= True)
data
`````

## Lo que hace es:

- df.Cancelled accede a la columna "Cancelled" de un DataFrame llamado "df", que probablemente contiene datos sobre vuelos.
- np.unique() es una funci√≥n de NumPy que encuentra todos los valores √∫nicos en un array.
- El par√°metro return_counts=True le dice a la funci√≥n que adem√°s de devolver los valores √∫nicos, tambi√©n cuente cu√°ntas veces aparece cada valor.

El resultado es una tupla con dos arrays:

- El primer array contiene los valores √∫nicos encontrados en la columna "Cancelled" (probablemente 0 y 1, donde 0 = no cancelado, 1 = cancelado)
- El segundo array contiene el conteo correspondiente para cada valor √∫nico

- Por ejemplo, si el resultado fuera ([0, 1], [9500, 500]), significar√≠a que hay 9500 vuelos no cancelados (0) y 500 vuelos cancelados (1).
- Este c√≥digo es com√∫nmente usado para an√°lisis exploratorio de datos o para preparar datos para visualizaciones como gr√°ficos de barras o circulares.

# ü•ß 5. plt.pie(valores) ‚Äì Gr√°fico circular (de pastel)

Muestra proporciones de una variable.

## üîß Par√°metros comunes:
- x: valores num√©ricos.
- labels: nombres de cada secci√≥n.
- colors: colores personalizados.
- autopct: formato para mostrar el porcentaje ('%1.1f%%').
- explode: desplaza una o m√°s secciones. # Ejemplo: explode = (0, 0.12, 0, 0, 0)  # Destacar el segundo elemento
- startangle: √°ngulo de inicio de la gr√°fica.

## Opciones de formato para autopct

## Formato b√°sico de porcentaje: '%1.1f%%'

- % indica que es un marcador de posici√≥n
- 1.1f significa un n√∫mero flotante con 1 d√≠gito antes y 1 despu√©s del punto decimal
- %% imprime el s√≠mbolo de porcentaje literal


## Variaciones comunes:

- '%1.0f%%' - Solo n√∫meros enteros (sin decimales)
- '%1.2f%%' - Dos decimales
- '%.0f%%' - Enteros sin especificar d√≠gitos antes del punto
