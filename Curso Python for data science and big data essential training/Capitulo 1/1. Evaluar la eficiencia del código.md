# Forma 2 con libreria datetime 
## Muestra el resultado de la siguiente forma: 0:00:01.000536

````

from datetime import timedelta

start = time.monotonic()

#nuestro codigo
time.sleep(1)

end = time.monotonic()

print(timedelta(seconds= end - start))

````
